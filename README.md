## Run Locally

Clone the project

```bash
  git clone https://github.com/Arcan7/Pierre-papier-ciseaux-python.git
```

Go to the project directory

```bash
  cd Pierre-papier-ciseaux-python
```

Run dependencies

```bash
  python main.py
```
Run 

```bash
  ./main
```

Run with cmd

```bash
  run.bat
```
